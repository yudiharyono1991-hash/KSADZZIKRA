# 01. Readme & Ringkasan Eksekutif
## BA Mart (Berkah Amanah Mart) v1.0 2026 IT Development

Selamat datang di repositori dokumentasi **BA Mart**, sebuah aplikasi Point of Sale (POS) dan Sistem Manajemen Keuangan mikro berbasis Syariah yang dirancang khusus untuk memenuhi kebutuhan operasional toko ritel staple goods (sembako) berskala UMKM di lingkungan kampus/lembaga syariah.

### Visi Utama Aplikasi
Membangun solusi kasir modern yang tidak hanya akurat secara matematis akuntansi, melainkan juga patuh secara penuh (*shariah compliant*) terhadap fatwa-fatwa perdagangan Dewan Syariah Nasional Majelis Ulama Indonesia (DSN-MUI). Melalui integrasi perhitungan Zakat Mal Perdagangan yang transparan, pelarangan ketat terhadap riba/bunga, penguncian harga rasi adil tanpa spekulasi, serta sinkronisasi database modern (Supabase), BA Mart menjadi pionir platform ritel mikro berbasis ESG (Environmental, Social, and Governance) di Indonesia tahun 2026.

### Peta Dokumen Teknis (8 Dokumen Rujukan Utama)
Folder `/docs` ini disusun secara khusus untuk memandu pengembang, pemilik modal, auditor syariah, dan operator dalam memahami serta mereplikasi sistem ini secara utuh di VS Code maupun lingkungan production:

1. **`01_Readme.md`**: Ringkasan eksekutif, visi misi, dan panduan peta dokumen teknis.
2. **`02_Architecture.md`**: Arsitektur teknis front-end React, Zustand State Management, dan skema tabel Supabase.
3. **`03_Compliance.md`**: Rukun Kepatuhan Syariah (Akad Murabahah, Fatwa MUI Zakat Dagang, & Kriteria ESG 1.0 2026).
4. **`04_AI_Spec.md`**: Spesifikasi integrasi AI, panduan prompt rekayasa instruksi, serta flow data.
5. **`05_Dev_Guide.md`**: Panduan deployment lokal di VS Code, instalasi dependensi, dan hot-reload server.
6. **`06_Security.md`**: Protokol keamanan data harian, otorisasi multi-user, audit trail, serta logs session.
7. **`07_Business_Rules.md`**: Logika bisnis operasional, rincian detail 3 role pengguna (Kasir, Admin, Owner) serta restriksinya.
8. **`08_UI_Guidelines.md`**: Panduan desain antarmuka, kombinasi palet warna hijau-emas, serta tata letak responsif.
